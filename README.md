### Secret Santa for Slack
With Node, Express, and Mongodb via Monk.

Built with the fine friends at [Pixel Union](http://pixelunion.net) in mind.

#### environment variables

```
SLACK_CLIENT_ID
SLACK_CLIENT_SECRET
SLACK_TEAM_ID
LOCAL_DB_URI
PORT = 3000
```

#### Running locally

```
$ npm install
```

```
$ gulp
```
